# Фотографии для слотов главной

Сейчас в слотах — фирменные SVG-заглушки внутри `<span class="illu">…</span>`
(файлы `src/partials/home.ru.html` и `home.en.html`). Чтобы поставить реальные
снимки, положите сюда файлы и замените содержимое `<span class="illu">` на `<img>`.

CSS уже готов: и `<svg>`, и `<img>` внутри слота растягиваются на весь слот с
`object-fit:cover`, так что замена — «один в один».

## Куда класть и как назвать

Слоты «Направления оборудования» (`.dir__img`, соотношение **16:10**, реком. **800×500**, @2x — 1600×1000):

| Файл | Раздел |
|---|---|
| `dir-diagnostics.jpg` | Диагностика и визуализация (УЗИ/рентген) |
| `dir-lab.jpg` | Лабораторное |
| `dir-surgery.jpg` | Хирургическое |
| `dir-icu.jpg` | Реанимация и ИВЛ |
| `dir-dental.jpg` | Стоматология |
| `dir-furniture.jpg` | Мебель и расходники |

Слоты «Реализованные поставки» (`.case__img`, соотношение **20:9**, реком. **1000×450**, @2x — 2000×900):

| Файл | Кейс |
|---|---|
| `case-acceptance.jpg` | Приёмка партии |
| `case-container.jpg` | Отгрузка контейнера |

Форматы: `.webp` или `.jpg` (webp легче). Оптимизируйте перед заливкой (squoosh.app,
или `cwebp`/`sharp`). Держите вес ≤ ~150 КБ на снимок.

## Как заменить SVG на `<img>`

В `src/partials/home.ru.html` (и точно так же в `home.en.html`) найдите слот, напр.:

```html
<div class="dir__img"><span class="illu"><svg ...>…длинный svg…</svg></span><span class="dir__tag">Фото: УЗИ / рентген</span></div>
```

и замените `<svg>…</svg>` на `<img>` (тег `<span class="dir__tag">` можно убрать —
это была подпись-подсказка «сюда фото»):

```html
<div class="dir__img"><span class="illu"><img src="/images/dir-diagnostics.jpg" alt="УЗИ-аппарат в клинике" width="800" height="500" loading="lazy" decoding="async"></span></div>
```

Для кейсов аналогично, но размеры `width="1000" height="450"` и файл `case-*.jpg`.

Подсказки:
- `alt` — короткое осмысленное описание (важно для SEO и доступности); на EN-версии
  дайте английский `alt`.
- `loading="lazy"` для всех, кроме самого верхнего — так страница грузится быстрее.
- Пути начинаются с `/images/…` (это папка `public/`, попадает в корень сайта).
